restrictionManagementEdit
directive in module restrictionManagementEditModule

Description

The restrictionManagementEdit Angular component is designed to be able to edit restrictions.

Usage

as element:
<restriction-management-edit
       [is-dirty-fn="{? Function}"]
       restriction="{< String}"
       [submit-fn="{? Function}"]
       uri-context="{< Object}"
       [get-supported-restriction-types-fn="{< Function}"]>
</restriction-management-edit>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
isDirtyFn
(optional)
? Function

Function returning the dirtiness status of the component.

restriction< String

Restriction object.

submitFn
(optional)
? Function

Function defined in outer scope to validate restriction edit.

uriContext< Object

The uriContext, as defined on the resourceLocationModule.

getSupportedRestrictionTypesFn
(optional)
< Function

A function that returns list of restriction types that are supported for editing for a given item.

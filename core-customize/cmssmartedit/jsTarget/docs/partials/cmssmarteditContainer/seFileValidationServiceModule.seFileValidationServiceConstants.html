seFileValidationServiceConstants
service in module seFileValidationServiceModule

Description

The constants provided by the file validation service.

ACCEPTED_FILE_TYPES: A list of file types supported by the platform. Default ACCEPTED_FILE_TYPES: ['jpeg', 'jpg', 'gif', 'bmp', 'tiff', 'tif', 'png', 'pdf'], Default MAX_FILE_SIZE_IN_BYTES: 20 1024 1024, Update supported file types to support pdf MAX_FILE_SIZE_IN_BYTES: The maximum size, in bytes, for an uploaded file. I18N_KEYS: A map of all the internationalization keys used by the file validation service.

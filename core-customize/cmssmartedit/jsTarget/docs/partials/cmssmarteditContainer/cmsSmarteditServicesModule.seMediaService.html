seMediaService
service in module cmsSmarteditServicesModule

Description

This service provides functionality to upload images and to fetch images by code for a specific catalog-catalog version combination.

Methods

  • uploadMedia(media)

    Uploads the media to the catalog.

    Parameters
    ParamTypeDetails
    mediaObject

    The media to be uploaded

    PropertyTypeDetails
    codeString

    A unique code identifier for the media

    descriptionString

    A description of the media

    altTextString

    An alternate text to be shown for the media

    fileFile

    The File object to be uploaded.

    Returns
    Promise<object>

    If request is successful, it returns a promise that resolves with the media object. If the request fails, it resolves with errors from the backend.

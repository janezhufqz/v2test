syncPageItem
directive in module pageComponentsModule

Description

syncPageItem builds a drop-down item allowing for the edition of a given CMS page .

Usage

as element:
<sync-page-item
       page-info="{<Object}">
</sync-page-item>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
pageInfo<Object

An object defining the context of the CMS page associated to the editPage item.

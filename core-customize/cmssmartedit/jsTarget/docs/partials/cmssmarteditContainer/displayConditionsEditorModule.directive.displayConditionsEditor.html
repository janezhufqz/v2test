displayConditionsEditor
directive in module displayConditionsEditorModule

Description

This component displays information about a page. For instance, it displays the page type, template, whether it's a primary or variation page, among others.

Usage

as element:
<display-conditions-editor
       page="{<Object}">
</display-conditions-editor>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
page<Object

The page for which to display its information

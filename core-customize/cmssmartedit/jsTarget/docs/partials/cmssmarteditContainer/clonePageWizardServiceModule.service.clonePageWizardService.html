clonePageWizardService
service in module clonePageWizardServiceModule

Description

The clone page wizard service allows opening a modal wizard to clone a page.

Methods

  • openClonePageWizard(pageData)

    When called, this method opens a modal window containing a wizard to clone an existing page.

    Parameters
    ParamTypeDetails
    pageDataObject

    An object containing the pageData when the clone page wizard is opened from the page list.

    Returns
    Promise

    A promise that will resolve when the modal wizard is closed or reject if it's canceled.

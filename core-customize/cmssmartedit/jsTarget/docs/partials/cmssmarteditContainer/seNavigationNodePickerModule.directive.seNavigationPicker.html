seNavigationPicker
directive in module seNavigationNodePickerModule

Description

Directive that will build a navigation node picker and assign the uid of the selected node to model[qualifier].

Usage

as element:
<se-navigation-picker
       uri-context="{Object}"
       model="{Object}"
       qualifier="{String}"
       [editable="{<Boolean}"]>
</se-navigation-picker>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
uriContextObject

the UriContext necessary to perform operations.

modelObject

the model a property of which will be set to the selected node uid.

qualifierString

the name of the model property that will be set to the selected node uid.

editable
(optional)
<Boolean

This property specifies whether the selector can be edited or not. If this flag is false, then the selector is treated as read-only; the selection cannot be modified in any way. Optional, default value is true.

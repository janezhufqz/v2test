navigationEditor
directive in module navigationEditorModule

Description

Navigation Editor directive used to display navigation editor tree

Usage

as element:
<navigation-editor
       uri-context="{Object}"
       read-only
       root-node-uid="{String}">
</navigation-editor>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
uriContextObject

the UriContext necessary to perform operations

readOnlyBoolean

when true, no CRUD facility shows on the editor. OPTIONAL, default false.

rootNodeUidString

the uid of the node to be taken as root, OPTIONAL, default "root"

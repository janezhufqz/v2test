deletePageItem
directive in module pageComponentsModule

Description

deletePageItem builds a dropdown item allowing for the soft deletion of a given CMS page .

Usage

as element:
<delete-page-item
       page-info="{< Object}">
</delete-page-item>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
pageInfo< Object

An object defining the context of the CMS page associated to the deletePage item.

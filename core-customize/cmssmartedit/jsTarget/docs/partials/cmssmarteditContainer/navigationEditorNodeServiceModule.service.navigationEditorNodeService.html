navigationEditorNodeService
service in module navigationEditorNodeServiceModule

Description

This service updates the navigation node by making REST call to the cmswebservices navigations API.

Methods

  • getNavigationNodeAncestry(node, uriParams)

    Returns the list of nodes belonging to the ancestry of the node identified by its uid. This list includes the queried node as well.

    Parameters
    ParamTypeDetails
    nodeObject

    The navigation node that needs to be updated.

    uriParamsObject

    the UriContext necessary to perform operations

    Returns
    Array

    an array of nodes

  • updateNavigationNodePosition(node)

    Updates the position of the navigation node within the children collection of its parent. Fetches the parent node, and reorders the children, then updates the parent with the new child order.

    Parameters
    ParamTypeDetails
    nodeObject

    The navigation node that needs to be updated.

restrictionManagementSelect
directive in module restrictionManagementSelectModule

Description

The restrictionManagementSelect Angular component is designed to be able to create or display restrictions.

Usage

as element:
<restriction-management-select
       [existing-restrictions="{< Array}"]
       [is-dirty-fn="{? Function}"]
       [submit-fn="{? Function}"]
       uri-context="{< Object}"
       get-restriction-types-fn="{& Expression}"
       [get-supported-restriction-types-fn="{< Function}"]>
</restriction-management-select>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
existingRestrictions
(optional)
< Array

Array of existing restrictions, that will be not be selectable.

isDirtyFn
(optional)
? Function

Function returning the dirtiness status of the component.

submitFn
(optional)
? Function

Function defined in outer scope to validate restriction edit.

uriContext< Object

The uriContext, as defined on the resourceLocationModule.

getRestrictionTypesFn& Expression

A function that returns list of restriction types for a given item.

getSupportedRestrictionTypesFn
(optional)
< Function

A function that returns list of restriction types that are supported for editing for a given item. If not provided, all types are assumed to be supported.

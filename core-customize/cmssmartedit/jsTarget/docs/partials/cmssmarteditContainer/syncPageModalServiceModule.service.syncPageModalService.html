syncPageModalService
service in module syncPageModalServiceModule

Description

Convenience service to open n synchronization modal window for a given page's data.

Methods

  • open(page, uriContext)

    Uses the modalService to open a modal.

    The editor modal is wired with a save and cancel button.

    The content of the synchronization modal is the synchronizationPanelModule.component:synchronizationPanel.

    Parameters
    ParamTypeDetails
    pageObject

    The data associated to a page as defined in the platform.

    uriContextObject

    A uriContext

    Returns
    Promise

    A promise that resolves to the data returned by the modal when it is closed.

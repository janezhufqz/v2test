seItemSelectorPanel
directive in module itemSelectorPanelModule

Description

Directive used to create a panel where a user can add items from one or more catalogs. Note that this directive is intended to be used within a seCatalogAwareSelector; it is not designed to work alone.

Usage

as element:
<se-item-selector-panel
       panel-title="{<String}"
       items-selected="{=Array}"
       item-template="{<String}"
       get-catalogs="{<Function}"
       items-fetch-strategy="{<Object}"
       on-change="{=Function}"
       show-panel="{=Function}"
       [hide-panel="{=Function}"]
       catalog-item-type="{<String}"
       [max-num-items="{<Number}"]>
</se-item-selector-panel>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
panelTitle<String

The title displayed at the top of the panel.

itemsSelected=Array

Array containing the UIDs of the items selected. There's a double binding, which means that the panel will show items already selected and will update back whenever the user changes the selection.

itemTemplate<String

The URL of the template used to display items in the component.

getCatalogs<Function

Function called by the component to retrieve the catalogs from which to select items from.

itemsFetchStrategy<Object

Object containing the strategy to retrieve items from the catalogs. There are three possible scenarios:

  • Provide a fetchAll function in the itemsFetchStrategy. This function must retrieve all items from the catalog at once. No pagination is performed.
  • Provide fetchPage + fetchEntity functions in the itemsFetchStrategy. Information about items already selected are retrieved one by one with fetchEntity. Items to choose from, on the other hand, are retrieved in a page fashion.
  • Provide fetchPage + fetchEntities Information about items already selected are retrieved in one call with fetchEntities. Items to choose from, on the other hand, are retrieved in a page fashion.
onChange=Function

Function executed whenever the selection of items change in the panel.

showPanel=Function

Function passed to the parent to be executed whenever the panel is opened.

hidePanel
(optional)
=Function

Function passed to the parent to be executed whenever the panel is hid.

catalogItemType<String

String specifying the type of elements being selected in the panel. This string is used only for localization purposes.

maxNumItems
(optional)
<Number

The maximum number of items that a user can select within the component.

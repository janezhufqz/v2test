clonePageItem
directive in module pageComponentsModule

Description

clonePageItem builds an item allowing for the cloning of a given CMS page.

Usage

as element:
<clone-page-item
       page-info="{< Object}">
</clone-page-item>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
pageInfo< Object

An object defining the context of the CMS page associated to the clonePage item.

componentEditingFacade
service in module componentEditingFacadeModule

Description

This service provides methods that allow adding or removing components in the page.

Methods

  • addExistingComponentToSlot(targetSlotId, dragInfo, position)

    This methods adds an existing component to the slot and display an Alert whenever the component is either hidden or restricted.

    Parameters
    ParamTypeDetails
    targetSlotIdString

    The ID of the slot where to drop the component.

    dragInfoObject

    The dragInfo object containing the componentId, componentUuid and componentType.

    positionNumber

    The position in the slot where to add the component.

  • addNewComponentToSlot(slotInfo, catalogVersionUuid, componentType, position)

    This methods adds a new component to the slot and opens a component modal to edit its properties.

    Parameters
    ParamTypeDetails
    slotInfoObject

    The target slot for the new component

    PropertyTypeDetails
    targetSlotIdObject

    The Uid of the slot where to drop the new component.

    targetSlotUUIdObject

    The UUid of the slot where to drop the new component.

    catalogVersionUuidString

    The catalog version on which to create the new component

    componentTypeString

    The type of the new component to add.

    positionNumber

    The position in the slot where to add the new component.

  • cloneExistingComponentToSlot(componentProperties)

    This methods clones an existing component to the slot by opening a component modal to edit its properties.

    Parameters
    ParamTypeDetails
    componentPropertiesObject

    The properties of the component required to create a clone

    PropertyTypeDetails
    targetSlotIdString

    The ID of the slot where to drop the component.

    dragInfoObject

    The dragInfo object containing the componentId, componentUuid and componentType.

    positionNumber

    The position in the slot where to add the component.

  • moveComponent(sourceSlotId, targetSlotId, componentId, position)

    This methods moves a component from two slots in a page.

    Parameters
    ParamTypeDetails
    sourceSlotIdString

    The ID of the slot where the component is initially located.

    targetSlotIdString

    The ID of the slot where to drop the component.

    componentIdString

    The ID of the component to add into the slot.

    positionNumber

    The position in the slot where to add the component.

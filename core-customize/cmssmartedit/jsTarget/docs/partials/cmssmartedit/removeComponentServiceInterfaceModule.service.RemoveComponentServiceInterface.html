RemoveComponentServiceInterface
service in module removeComponentServiceInterfaceModule

Description

Service interface specifying the contract used to remove a component from a slot.

This class serves as an interface and should be extended, not instantiated.

Methods

  • removeComponent(slotId, componentId)

    Removes the component specified by the given ID from the component specified by the given ID.

    Parameters
    ParamTypeDetails
    slotIdString

    The ID of the slot from which to remove the component.

    componentIdString

    The ID of the component to remove from the slot.

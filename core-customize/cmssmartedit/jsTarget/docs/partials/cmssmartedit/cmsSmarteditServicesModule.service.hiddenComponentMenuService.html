hiddenComponentMenuService
service in module cmsSmarteditServicesModule

Description

This service is used to retrieve menu items that are available to be used with hidden components.

Methods

  • allowItemsInHiddenComponentMenu(itemsToAllow)

    This method is used to set the list of items that can be used with hidden components.

    Parameters
    ParamTypeDetails
    itemsToAllowString[]

    The ID of the menu items that can be used with hidden components.

  • getItemsForHiddenComponent(component, slotId)

    This method is used to retrieve the menu items available to be used in the provided component. To do so, this method retrieves contextual menu items available for the provided component and filters out the ones that cannot be used in hidden components. For example, assuming that a visible component has 'drag and drop' and 'remove' contextual menu items, if the component is hidden it should only have the remove button available, since the drag and drop operation is meaningless if the component is hidden. Hence, this service will retrieve only the remove item.

    Parameters
    ParamTypeDetails
    componentObject

    The hidden component for which to retrieve its menu items.

    slotIdString

    The SmartEdit id of the slot where the component is located.

    Returns
    Promise<ContextualMenuItem[]>

    Promise that resolves to an array of contextual menu items available for the component provided.

  • removeAllowedItemsInHiddenComponentMenu(itemsToAllow)

    This method removes a provided set of allowed menu items if previously allowed.

    Parameters
    ParamTypeDetails
    itemsToAllowString[]

    An array containing the ID's of the menu items that cannot be used any longer with hidden components.

  • removeAllowedItemsInHiddenComponentMenu()

    This method retrieves the list of IDs of the menu items that can be used with hidden components.

    Returns
    String[]

    The list of IDs of the menu items that can be used with hidden components.

slotContainerService
service in module cmsSmarteditServicesModule

Description

This service allows retrieving information about the containers found in a given page.

Methods

  • getComponentContainer(slotId, componentUuid)

    This method is used to retrieve the information about the container holding the provided component. If the component is not inside a container, the method returns null.

    Parameters
    ParamTypeDetails
    slotIdString

    The SmartEdit id of the slot where the component in question is located.

    componentUuidString

    The UUID of the component as defined in the database.

    Returns
    angular.IPromise<ContainerInfo>

    A promise that resolves to the information of the container of the component provided. Will be null if the component is not inside a container.

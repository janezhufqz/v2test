pageListSyncIcon
directive in module pageListSyncIconModule

Description

The Page Synchronization Icon component is used to display the icon that describes the synchronization status of a page.

Usage

as element:
<page-list-sync-icon
       page-id="{string}">
</page-list-sync-icon>

Parameters

ParamTypeDetails
pageIdstring

The identifier of the page for which the synchronzation status must be displayed.

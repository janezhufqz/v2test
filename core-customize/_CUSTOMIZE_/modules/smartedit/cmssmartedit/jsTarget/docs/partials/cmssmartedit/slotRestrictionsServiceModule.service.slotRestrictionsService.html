slotRestrictionsService
service in module slotRestrictionsServiceModule

Description

This service provides methods that cache and return the restrictions of a slot in a page. This restrictions determine whether a component of a certain type is allowed or forbidden in a particular slot.

Methods

  • getAllComponentTypesSupportedOnPage()

    This methods retrieves the list of component types droppable in at least one of the slots of the current page

    Returns
    Promise

    A promise containing an array with the component types droppable on the current page

  • getSlotRestrictions(slotId)

    This methods retrieves the list of restrictions applied to the slot identified by the provided ID.

    Parameters
    ParamTypeDetails
    slotIdString

    The ID of the slot whose restrictions to retrieve.

    Returns
    Promise

    A promise containing an array with the restrictions applied to the slot.

  • isComponentAllowedInSlot(slot, dragInfo)

    This methods determines whether a component of the provided type is allowed in the slot.

    Parameters
    ParamTypeDetails
    slotObject

    the slot for which to verify if it allows a component of the provided type.

    PropertyTypeDetails
    idString

    The ID of the slot.

    componentsArray

    the list of components contained in the slot, they must contain an "id" property.

    dragInfoObject

    contains the dragged object information

    PropertyTypeDetails
    componentTypeString

    The smartedit type of the component being checked.

    componentIdString

    The smartedit id of the component being checked.

    slotIdString

    The smartedit id of the slot from which the component originates

    cloneOnDropString

    The boolean that determines if the component should be cloned or not

    Returns
    Promise

    A promise containing COMPONENT_IN_SLOT_STATUS (ALLOWED, DISALLOWED, MAYBEALLOWED) string that determines whether a component of the provided type is allowed in the slot.

  • isSlotEditable(slotId)

    This method determines whether slot is editable or not.

    Parameters
    ParamTypeDetails
    slotIdString

    The ID of the slot.

    Returns
    Promise

    A promise containing a boolean flag that shows whether if the slot is editable or not.

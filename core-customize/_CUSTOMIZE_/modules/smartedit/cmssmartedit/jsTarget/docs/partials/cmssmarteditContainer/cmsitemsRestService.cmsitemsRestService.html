cmsitemsRestService
service in module cmsitemsRestService

Description

Service to deal with CMS Items related CRUD operations.

Methods

  • create(cmsitem)

    Create a new CMS Item.

    Parameters
    ParamTypeDetails
    cmsitemObject

    The object representing the CMS Item to create

    Returns
    Promise<CMSItem>

    If request is successful, it returns a promise that resolves with the CMS Item object. If the request fails, it resolves with errors from the backend.

  • delete(cmsitemUuid)

    Remove a CMS Item.

    Parameters
    ParamTypeDetails
    cmsitemUuidNumber

    The CMS Item uuid

  • get(queryParams)

    Fetch CMS Items search result by making a REST call to the CMS Items API. A search can be performed by a typeCode (optionnaly in combination of a mask parameter), or by providing a list of cms items uuid.

    Parameters
    ParamTypeDetails
    queryParamsObject

    The object representing the query params

    PropertyTypeDetails
    pageSizeString

    number of items in the page

    currentPageString

    current page number

    typeCode
    (optional)
    		String

    for filtering on the cms item typeCode

    mask
    (optional)
    		String

    for filtering the search

    itemSearchParams
    (optional)
    		String

    search on additional fields using a comma separated list of field name and value pairs which are separated by a colon. Exact matches only.

    catalogId
    (optional)
    		String

    the catalog to search items in. If empty, the current context catalog will be used.

    catalogVersion
    (optional)
    		String

    the catalog version to search items in. If empty, the current context catalog version will be used.

    Returns
    Promise<CMSItem>

    If request is successful, it returns a promise that resolves with the paged search result. If the request fails, it resolves with errors from the backend.

  • getByIds(uuids)

    Fetch CMS Items by uuids, making a POST call to the CMS Items API. A search can be performed by providing a list of cms items uuid.

    Parameters
    ParamTypeDetails
    uuids
    (optional)
    		string[]

    list of cms item uuids

    Returns
    Promise<CMSItem[]>

    If request is successful, it returns a promise that resolves to the result. If the request fails, it resolves with errors from the backend. Be mindful that the response payload size could increase dramatically depending on the number of uuids that you send on the request.

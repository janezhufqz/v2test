actionableAlertService
service in module actionableAlertModule

Description

The actionableAlertService is used by external modules to render an Alert structured around a description, an hyperlink label and a custom controller.

Methods

  • displayActionableAlert(alertContent, alertType)

    Parameters
    ParamTypeDetails
    alertContentObject

    A JSON object containing the specific configuration to be applied on the actionableAlert.

    PropertyTypeDetails
    controllerFunction

    Function defining Angular controller consumed by the contextual alert.

    descriptionString

    Description displayed within the contextual alert.

    hyperlinkLabelString

    Label for the hyperlink displayed within the contextual alert.

    timeoutDurationNumber?

    The timeout duration of the cms alert item, in milliseconds. If not provided, the default is used.

    alertTypeString?

    A string representing the type of alert to display. The string must match one of the types defined in ALERT_TYPES. If the alert type is null, or it doesn't match any of the valid types this method will default to INFO.

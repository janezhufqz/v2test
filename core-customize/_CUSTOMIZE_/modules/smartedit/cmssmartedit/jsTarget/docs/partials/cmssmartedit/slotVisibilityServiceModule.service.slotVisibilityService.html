slotVisibilityService
service in module slotVisibilityServiceModule

Description

The slotVisibilityService provides methods to reload slot information and manage hidden components.

Methods

  • getHiddenComponents(slotId)

    Returns the list of hidden components for a given slotId

    Parameters
    ParamTypeDetails
    slotIdString

    the slot id

    Returns
    Promise

    A promise that resolves to a list of hidden components for the slotId

  • reloadSlotsInfo()

    Reloads and cache's the pagesContentSlotsComponents for the current page in context. this method can be called when ever a component is added or modified to the slot so that the pagesContentSlotsComponents is re-evalated.

    Returns
    Promise

    A promise that resolves to the contentSlot - Components [] map for the page in context.

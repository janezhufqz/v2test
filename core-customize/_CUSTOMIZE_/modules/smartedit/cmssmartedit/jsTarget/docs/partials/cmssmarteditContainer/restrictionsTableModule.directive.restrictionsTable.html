restrictionsTable
directive in module restrictionsTableModule

Description

Directive that can render a list of restrictions and provides callback functions such as onSelect and onCriteriaSelected. *

Usage

as element:
<restrictions-table
       custom-class="{= String}"
       editable="{= Boolean}"
       [on-click-on-edit="{< Function}"]
       on-select="{= Function}"
       restrictions="{= Object}"
       [restriction-criteria="{= Object }"]
       [errors="{< Array }"]>
</restrictions-table>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
customClass= String

The name of the CSS class.

editable= Boolean

States whether the restrictions table could be modified.

onClickOnEdit
(optional)
< Function

Triggers the custom on edit event.

onSelect= Function

Triggers the custom on select event.

restrictions= Object

The object of restrictions.

restrictionCriteria
(optional)
= Object

The object that contains information about criteria.

errors
(optional)
< Array

The list of errors.

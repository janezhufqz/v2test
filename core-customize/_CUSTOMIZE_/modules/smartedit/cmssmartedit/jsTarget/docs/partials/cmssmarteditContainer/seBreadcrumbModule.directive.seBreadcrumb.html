seBreadcrumb
directive in module seBreadcrumbModule

Description

Directive that will build a navigation breadcrumb for the Node identified by either uuid or uid.

Usage

as element:
<se-breadcrumb
       [node-uid="{String}"]
       [node-uuid="{String}"]
       uri-context="{Object}">
</se-breadcrumb>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
nodeUid
(optional)
String

the uid of the node the breadcrumb of which we want to build.

nodeUuid
(optional)
String

the uuid of the node the breadcrumb of which we want to build.

uriContextObject

the UriContext necessary to perform operations.

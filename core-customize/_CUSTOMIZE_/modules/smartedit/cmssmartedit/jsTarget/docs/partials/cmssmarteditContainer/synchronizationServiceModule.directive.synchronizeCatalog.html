synchronizeCatalog
directive in module synchronizationServiceModule

Description

The synchronize catalog directive is used to display catalog synchronization information in the landing page.

For the active-catalog version, this directive displays the information of the last sync job affecting that catalog version.

For non-active catalog versions it provides information about the last sync job from that catalog version towards the catalog's active version. Also, it displays a button to trigger a new synchronization job, going from the current catalog version to the catalog's active version.

Usage

as element:
<synchronize-catalog
       catalog="{Object}"
       catalog-version="{Object}"
       active-catalog-version="{Object}">
</synchronize-catalog>

Parameters

ParamTypeDetails
catalogObject

An object that contains the catalog details.

catalogVersionObject

An object representing the current catalog version.

activeCatalogVersionObject

An object representing the active catalog version of the provided catalog.

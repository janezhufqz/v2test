componentCloneInfoForm
directive in module componentCloneInfoFormModule

Description

Component for the clone page info form

Usage

as element:
<component-clone-info-form
       structure="{<Object}"
       content="{<Object}"
       submit="{Function}"
       reset="{Function}"
       is-dirty="{Function}"
       is-valid="{Function}"
       page-template="{<String}"
       page-type-code="{<String}"
       uri-context="{<Object}"
       target-catalog-version="{<Object}">
</component-clone-info-form>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
structure<Object

The structure that is passed on to the generic editor

content<Object

The model that is passed on to the generic editor

submitFunction

Function defined in outer scope for saving the form fields

resetFunction

Function defined in outer scope for reseting all fields in the form

isDirtyFunction

Function defined in outer scope returning if the form is in pristine state or not (e.g., have been modified).

isValidFunction

Function defined in outer scope returning if all fields in the form are valid

pageTemplate<String

The pageTemplate property of the cloned page

pageTypeCode<String

The typeCode property of the cloned page

uriContext<Object

The uri context containing site/catalog information. This is necessary for the component to determine if the selected target catalog version is different then the current page context

PropertyTypeDetails
siteUIDString

The site ID of the current page

catalogIdString

The catalog ID of the current page

catalogVersionString

The catalog version of the current page

targetCatalogVersion<Object

The selected catalogVersion containing the catalogId, version, and siteId

PropertyTypeDetails
siteIdString

The selected site ID

catalogIdString

The selected catalog ID

versionString

The selected catalog version

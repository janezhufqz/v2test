seFileValidationService
service in module seFileValidationServiceModule

Description

The seFileValidationService validates that the file provided is of a specified file type and that the file does not exceed the maxium size limit for the file's file type.

Methods

  • buildAcceptedFileTypesList()

    Transforms the supported file types provided by the seFileValidationServiceConstants service into a comma- separated list of file type extensions.

    Returns
    String

    A comma-separated list of supported file type extensions

  • validate(file, context)

    Validates the specified file object using validator provided by the seFileObjectValidators and the file header validator provided by the seFileMimeTypeService. It appends the errors to the error context array provided or it creates a new error context array.

    Parameters
    ParamTypeDetails
    fileFile

    The web API file object to be validated.

    contextArray

    The contextual error array to append the errors to. It is an output parameter.

    Returns
    Promise

    A promise that resolves if the file is valid or a list of errors if the promise is rejected.

pageTemplateService
service in module pageTemplateServiceModule

Description

This service allows the retrieval of page templates associated to a page type.

Methods

  • getPageTemplatesForType(uriContext, pageType)

    When called, this method retrieves the page templates associated to the provided page type.

    Parameters
    ParamTypeDetails
    uriContextObject

    A UriContext

    pageTypeString

    The page type for which to retrieve its associated page templates.

    Returns
    Promise

    A promise that will resolve with the page templates retrieved for the provided page type.

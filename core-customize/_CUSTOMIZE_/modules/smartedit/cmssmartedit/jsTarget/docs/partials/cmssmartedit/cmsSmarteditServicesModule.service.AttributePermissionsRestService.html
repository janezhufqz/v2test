AttributePermissionsRestService
service in module cmsSmarteditServicesModule

Description

Rest Service to retrieve attribute permissions.

Methods

  • hasChangePermissionOnAttributesInType(type, attributeNames)

    Determines if the current user has CHANGE access to the given attributes in the given type.

    Parameters
    ParamTypeDetails
    typeString

    The type enclosing the attributes for which to evaluate their change permissions.

    attributeNamesString[]

    The names of the attributes for which to evaluate their change permissions.

    Returns
    angular.IPromise<TypedMap<boolean>>

    A promise that resolves to a TypedMap object with key (the attribute name) and value (true if the user has READ access to the type or false otherwise).

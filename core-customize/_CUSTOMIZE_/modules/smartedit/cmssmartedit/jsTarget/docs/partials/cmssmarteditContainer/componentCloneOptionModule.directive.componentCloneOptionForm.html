componentCloneOptionForm
directive in module componentCloneOptionModule

Description

Component for selecting the clone page options for child components in content slots. The options are clone existing child components or reference them

Usage

as element:
<component-clone-option-form
       on-selection-change="{&Expression}">
</component-clone-option-form>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
onSelectionChange&Expression

The parent function to call when a selection is made.

selectPageTemplate
directive in module selectPageTemplateModule

Description

Displays a list of all CMS page templates in the system, and allows the user to select one, triggering the on-template-selected callback.

Usage

as element:
<select-page-template
       on-template-selected="{Function}">
</select-page-template>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
onTemplateSelectedFunction

[Required] A callback function that is called when a template is selected from the list. The function is called with a single argument, an object representing the selected page template.

pageListLink
directive in module pageListLinkModule

Description

Directive that displays a link to the list of pages of a catalogue. It can only be used if catalog.catalogVersions is in the current scope.

Usage

as element:
<page-list-link
       catalog="{< Object}"
       catalog-version="{< Boolean}"
       site-id="{< String}">
</page-list-link>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
catalog< Object

Object representing the provided catalog.

catalogVersion< Boolean

Object representing the provided catalog version.

siteId< String

The ID of the site the provided catalog is associated with.

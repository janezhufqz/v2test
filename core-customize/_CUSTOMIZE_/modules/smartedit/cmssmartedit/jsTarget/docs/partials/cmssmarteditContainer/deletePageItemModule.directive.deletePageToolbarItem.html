deletePageToolbarItem
directive in module deletePageItemModule

Description

deletePageToolbarItem provides a tooolbar action to Move a page to trash.

Usage

as element:
<delete-page-toolbar-item
       toolbar-item="{<Object}">
</delete-page-toolbar-item>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
toolbarItem<Object

An object that represents the toolbar item.

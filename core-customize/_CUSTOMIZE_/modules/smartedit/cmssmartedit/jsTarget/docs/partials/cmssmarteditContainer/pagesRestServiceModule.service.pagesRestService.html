pagesRestService
service in module pagesRestServiceModule

Description

The pagesRestService provides core REST functionality for the CMS pages rest endpoint

Methods

  • get(params)

    Fetches a list of pages for a given site, catalog, and catalog version. If the site, catalog, or catalog version is not defined, those used contextually in the session will be used instead.

    Parameters
    ParamTypeDetails
    paramsObject

    A JSON object containing catalog context and/or any additional request parameters

    PropertyTypeDetails
    siteUIDString

    A side ID

    catalogIdString

    A catalog ID

    catalogVersionString

    A catalog version ID

    Example:

    {
     siteUID: 'supershoes',
     catalogId: 'shoes',
     catalogVersion: 'online',
     anOptionalQueryParamName: 'paramValue'
}
    Returns
    Promise<Array>

    A promise resolving to a list of pages, or an empty list.

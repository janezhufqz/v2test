seProductSelector
directive in module productSelectorModule

Description

A component that allows users to select products from one or more catalogs. This component is catalog aware; the list of products displayed is dependent on the product catalog and catalog version selected by the user within the component.

Usage

as element:
<se-product-selector
       id="{@String}"
       model="{<Object}"
       qualifier="{<String}"
       [editable="{<Boolean}"]>
</se-product-selector>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
id@String

An identifier used to track down the component in the page.

model<Object

The object where the list of selected products will be stored. The model must contain a property with the same name as the qualifier. That property must be of type array and is used to store the UIDs of the products selected.

qualifier<String

The key of the property in the model that will hold the list of products selected.

editable
(optional)
<Boolean

This property specifies whether the selector can be edited or not. If this flag is false, then the selector is treated as read-only; the selection cannot be modified in any way. Optional, default value is true.

WorkflowTasksPollingService
service in module cmsSmarteditServicesModule

Description

This service is used to retrieve inbox tasks.

Methods

  • addSubscriber(subscriber, callOnInit)

    Adds a new subscriber to the polling service. The subscriber is called with a list of new tasks and a pagination information.

    Parameters
    ParamTypeDetails
    subscriberWorkflowTaskSubscriber

    the subscriber.

    callOnInitboolean

    Default is true, when set to false, will not call the subscriber on initialization of the polling.

    Returns

    the method that can be used to unsubscribe.

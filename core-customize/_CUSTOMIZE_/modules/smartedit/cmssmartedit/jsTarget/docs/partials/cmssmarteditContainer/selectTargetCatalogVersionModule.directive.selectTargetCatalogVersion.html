selectTargetCatalogVersion
directive in module selectTargetCatalogVersionModule

Description

Displays a list of catalog versions used for selecting the target catalog version that will be applied to the cloned page

Usage

as element:
<select-target-catalog-version
       uri-context="{< Object}"
       page-type-code="{< String}"
       page-label="{< String}"
       [on-target-catalog-version-selected="{< Function }"]>
</select-target-catalog-version>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
uriContext< Object

The uri context containing site/catalog information. This is necessary for the component to determine the list of catalog versions to display

PropertyTypeDetails
siteUIDString

The site ID for the new page

catalogIdString

The catalog ID for the new page

catalogVersionString

The catalog version for the new page

pageTypeCode< String

The page typeCode of a potential new page

pageLabel< String

The page label of a potential new page

onTargetCatalogVersionSelected
(optional)
< Function

An optional output function binding. Every time there is a change to the catalogVersion selection, or resulting target catalog version, this function (if it exists) will get executed

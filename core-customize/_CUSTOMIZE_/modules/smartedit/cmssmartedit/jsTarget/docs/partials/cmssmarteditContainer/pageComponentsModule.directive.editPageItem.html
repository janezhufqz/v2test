editPageItem
directive in module pageComponentsModule

Description

editPageItem builds an action item allowing for the edition of a given CMS page .

Usage

as element:
<edit-page-item
       page-info="{< Object}">
</edit-page-item>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
pageInfo< Object

An object defining the context of the CMS page associated to the editPage item.

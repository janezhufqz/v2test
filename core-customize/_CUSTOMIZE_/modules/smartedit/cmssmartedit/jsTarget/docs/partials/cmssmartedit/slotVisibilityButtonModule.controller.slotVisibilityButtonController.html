slotVisibilityButtonController
controller in module slotVisibilityButtonModule

Description

The slot visibility button controller is responsible for enabling and disabling the hidden components button, as well as displaying the hidden components list. It also provides functions to open and close the hidden component list.

Usage

slotVisibilityButtonController(slotVisibilityService, $scope);

Parameters

ParamTypeDetails
slotVisibilityServiceObject

slot visibility service instance

$scopeObject

current scope instance
